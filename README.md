# devops-tp

API REST de tareas construida con Node.js, Express, TypeScript y PostgreSQL (Prisma). El pipeline de CI/CD corre en GitHub Actions: ejecuta los tests, construye la imagen Docker multi-stage, la publica en Docker Hub y dispara el deploy automático en Render; el monitoreo de la aplicación en producción se realiza con OpenTelemetry y Grafana.
